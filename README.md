# 📧 COLD-EMAIL-GENERATION-TOOL

An AI-powered application that generates **personalized cold emails** for job applications, networking, internships, and business outreach. Built with **Python, LangChain, Groq LLM, and Streamlit**, it creates professional emails in seconds based on user input.

## 🚀 Features
- AI-generated personalized cold emails
- Professional and customizable tone
- Simple Streamlit interface
- Fast and easy to use
- Powered by LangChain and Groq LLM

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- Groq LLM
- Prompt Engineering
- dotenv

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/COLD-EMAIL-GENERATION-TOOL.git
cd COLD-EMAIL-GENERATION-TOOL
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
```

Run the app:

```bash
streamlit run app.py
```
